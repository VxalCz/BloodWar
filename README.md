# BloodWar

Vampire Survivors clone in Python.

## Install

```bash
pip install pygame-ce
```

## Run

```bash
python main.py
```

## Controls

- **WASD** - Move player
- **G** - Toggle debug grid
- **R** - Restart (after game over)

## Features

- Enemy spawning and chasing
- Magic Wand auto-shooting
- Experience gems with magnetic pickup
- Score system
- Y-sorted rendering

## Tech

- Python + pygame-ce
- Modular architecture (src/)
